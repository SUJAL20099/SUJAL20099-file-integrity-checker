# File Integrity Checker

## Overview
A cybersecurity web application that verifies file integrity using cryptographic hash algorithms such as SHA-256 and MD5.

## Features
- Upload files securely
- Generate SHA-256, SHA-1, and MD5 hashes
- Compare hash values
- Detect file tampering
- User-friendly web interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Render
- GitHub
